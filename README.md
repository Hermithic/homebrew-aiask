# Homebrew Tap for AIask

This is the official Homebrew tap for [AIask](https://github.com/Hermithic/aiask), an AI-powered command line assistant.

## Installation

```bash
brew tap Hermithic/aiask
brew install aiask
```

## Usage

After installation, configure your LLM provider:

```bash
aiask config
```

Then use it:

```bash
aiask "list all files larger than 100MB"
```
